Overview:

Hex Allen Key 10 Piece Set Imperial HK19 a handy set for working with fasteners that use imperial (SAE) hex sockets. Ideal for machinery, older equipment, bikes and imported fixings, these keys are made from durable chrome vanadium steel for reliable everyday use. Supplied in a practical storage clip with a hanging hole, the set stays organised and easy to carry or store on a hook or pegboard. Imperial sizes are clearly marked on the holder to help you select the correct key quickly and return it to the right position after use.

How to Use

  • Choose the correct size key to avoid rounding the socket

  • Insert the key fully into the fastener before applying force

  • Turn steadily and keep the key aligned with the fastener

  • If the fastener is tight, apply controlled pressure rather than sudden force

  • Clean the keys after use and return them to the marked holder

  • Store dry and keep the set together to prevent lost sizes
